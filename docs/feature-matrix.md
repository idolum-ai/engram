# Engram Feature Matrix

This table tracks user-facing and operational features. The "tested %" column is
an engineering confidence estimate from current automated and live/manual
coverage, not Go statement coverage.

Readiness levels:

- `ready`: implemented, exercised by `make check`, and usable in the current service.
- `beta`: implemented and usable, but relies meaningfully on live Telegram/tmux/LLM behavior.
- `partial`: implemented with known coverage or behavior gaps.
- `reserved`: intentionally exposed in metadata but not meant for normal use.
